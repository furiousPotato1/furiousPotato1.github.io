AR Setup Required

WebGL Support

Required for 3D rendering. Currently {{ webglSupported ? 'enabled' : 'disabled' }}

To enable on Android:

  1. Open Chrome and type chrome://flags
  2. Search for "WebGL"
  3. Enable "WebGL 2.0" and "WebGL Draft Extensions"
  4. Restart Chrome

To enable on iOS:

  1. Go to Settings → Safari → Advanced
  2. Toggle "WebGL" to ON

WebRTC Support

Required for camera access. Currently {{ webrtcSupported ? 'enabled' : 'disabled' }}

To enable on Android:

  1. Open Chrome and type chrome://flags
  2. Search for "WebRTC"
  3. Enable "WebRTC hardware video encoding"
  4. Restart Chrome

GPU Acceleration

Recommended for better performance

To enable on Android:

  1. Open Chrome and type chrome://flags
  2. Search for "GPU"
  3. Enable "GPU rasterization" and "Accelerated 2D canvas"
  4. Restart Chrome
